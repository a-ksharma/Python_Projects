📈 Compound Interest Calculator
This is a simple Compound Interest Calculator program written in Python. It helps users compute the future value of an investment based on the principal amount, interest rate, compounding frequency, and time.

🚀 Features
Calculates compound interest using user inputs

Easy-to-use command-line interface

Supports different compounding frequencies (annually, semi-annually, quarterly, monthly, daily)

Provides a clear breakdown of the results

🧮 Formula Used
The program uses the standard compound interest formula:

𝐴=𝑃×(1+𝑟/𝑛)^𝑛x𝑡
 
Where:

A = Final amount (including interest)

P = Principal amount (initial investment)

r = Annual interest rate (in percentage)

n = Number of times interest is compounded per year

t = Number of years

main.py: Main Python script for the calculator.
