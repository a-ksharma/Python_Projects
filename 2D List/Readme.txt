# 2D List Creation and Traversal in Python

## Description

-This Python script allows a user to dynamically create a 2D list (list of lists) by specifying the number of rows and columns. The user is prompted to input each element of the list. The script then displays the full 2D list and traverses it row by row in a structured format.

## Features

- Accepts user input for dimensions and elements of the list.
- Dynamically builds and stores the list.
- Nicely prints and traverses the 2D list.
- Good for beginners learning nested loops and 2D data structures.

## How to Run

1. Make sure you have Python 3 installed.
2. Save the script as `main.py`.
3. Run the script using the terminal or command prompt: "python main.py:
